# Từ điển thuật ngữ

## Quy tắc
- Không dịch tùy hứng.
- Dùng đúng ánh xạ dưới đây trong tài liệu.
- Code, biến, hàm, class giữ tiếng Anh.

## Ánh xạ chuẩn
- problem statement -> đề bài
- constraints -> ràng buộc
- sample input -> ví dụ đầu vào
- sample output -> ví dụ đầu ra
- crawler -> crawler
- queue -> queue
- pipeline -> pipeline
- compile -> biên dịch
- template -> template
- archive -> archive
- metadata -> metadata
- report -> báo cáo
- audit -> audit
- policy -> chính sách
